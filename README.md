# Python Lists and Conditional Statements

## Description

This program demonstrates the use of Python lists, list comprehension, inventory management, and conditional statements such as if, elif, and else. It also includes programs to check whether a number is even or odd, assign grades based on marks, and find the largest of three numbers.

## Concepts Used

- Lists
- List Creation
- List Comprehension
- Range Function
- String to List Conversion
- List Operations
  - Concatenation
  - Repetition
  - Slicing
  - Deletion
- Inventory Management using List Comprehension
- User Input
- Conditional Statements
  - if
  - elif
  - else
- Even/Odd Number Check
- Grade Calculation
- Largest of Three Numbers
